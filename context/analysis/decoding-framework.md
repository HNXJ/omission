# Omission Decoding Methodology & Hypothesis Framework

## 🎯 Overview
This document outlines the systematic decoding pipeline used to quantify the "Neural Surprise" and "Contextual Persistence" in a sequential visual omission task. We utilize multi-modal signals (Single-Units, LFP Channels, Pupil/Eye) to decode stimulus identity and omission presence.

## ⚙️ Decoding Protocol
- **Classifier**: Linear Support Vector Machine (SVM).
- **Validation**: 5-Fold Stratified Cross-Validation.
- **Balancing**: 50/50 Random Downsampling (Equal trials per class).
- **Normalization**: Z-score across features.
- **Performance Targets**: 
  - **Train Accuracy**: Target > 90% (to ensure separability).
  - **Test Accuracy**: Above 0.5 (Chance) indicates a statistically significant signal.

## 🔬 Core Hypothesis Framework
We systematically test 108 combinations across 9 task intervals (FX, P1-4, D1-4) and 12 condition variants.

### 1. Identity Decoding (A vs. B)
**Tests**: `p1-of-aaab` vs. `p1-of-bbba`, `p2-of-aaab` vs. `p2-of-bbba`, etc.
- **Interpretation**: If `p1` decoding is > chance, the population encodes the *current* stimulus identity.
- **Wait... Contextual Encoding**: If `fx` (Fixation) decoding is > chance, the population encodes the *block-level expectation* before stimulus onset.

### 2. Omission Detection (Something vs. Nothing)
**Tests**: `p4-of-aaax` vs. `p4-of-aaab`, `p3-of-aaxb` vs. `p3-of-aaab`, etc.
- **Interpretation**: Measures the detection of a "Ghost" signal where visual input is absent (Omission) compared to present (Standard).

### 3. Within-Trial Omission (Omit vs. Delay)
**Tests**: `p4-of-aaax` vs. `d3-of-aaax`, `p3-of-aaxb` vs. `d2-of-aaxb`, etc.
- **Interpretation**: The **Rigorous Null Hypothesis**. Since the screen is physically the same (Gray) in both windows, any decoding performance > chance reflects a purely internal state change triggered by the *expected* stimulus onset time.

### 4. Omission Type/Context (e.g., `p3-of-bbxa` vs. `p3-of-aaxb`)
- **Significance**: If this gets above chance performance, it means the brain's "Surprise" signal is **identity-specific**. It's not just "Something is missing"; it's "B is missing" vs. "A is missing." 
- This would imply that the prediction error carries the identity of the expected (but absent) stimulus.

## 📊 Feature Space
1.  **Single-Unit (Individual)**: Decoding accuracy is computed for *each neuron independently* to identify "Information Hubs."
2.  **LFP Channel (Individual)**: Decoding accuracy per electrode (1-128) to map the **Information-Rich Layers** (L4 vs. L5/6).
3.  **Pupil/Eye**: Behavioral proxies for global arousal or surprise.

## 🧪 The 108 Decoding Matrix (Hypothesis Space)

We systematically iterate through **9 Intervals** (Fixation, P1, D1, P2, D2, P3, D3, P4, D4) and **12 Condition Pairings**. This logic is implemented in `scripts/decode_omission_identity.py` and `scripts/run_individual_decoding.py`.

### Identity Contrast (A vs. B) - 18 Tests
- `[Interval]-of-AAAB` vs. `[Interval]-of-BBBA` (9 intervals)
- `[Interval]-of-AAAX` vs. `[Interval]-of-BBBX` (9 intervals)
- **Significance**: If A vs B is decodable during `fx` (Fixation), the brain maintains a **Pre-Stimulus Contextual Expectation**.
- **Script**: `scripts/run_figure_01_classification.py` (Pupil) and `scripts/run_figure_03_classification_neural.py` (Spikes).

### Omission Contrast (Omit vs. Std) - 18 Tests
- `p2-of-AXAB` vs. `p2-of-AAAB` (9 intervals for sequence matching)
- `p3-of-AAXB` vs. `p3-of-AAAB`
- `p4-of-AAAX` vs. `p4-of-AAAB`
- **Significance**: Quantifies the "Ghost Signal" amplitude relative to sensory-driven activity.
- **Script**: `scripts/run_figure_02_classification_combined.py`.

### Surprise Identity (Identity of the Missing) - 18 Tests
- `p2-of-AXAB` vs. `p2-of-BXBA`
- `p3-of-AAXB` vs. `p3-of-BBXA`
- `p4-of-AAAX` vs. `p4-of-BBBX`
- **Significance**: If `p3-of-BBXA` vs. `p3-of-AAXB` gets above chance performance, the brain's surprise is **identity-specific**. It indicates the prediction error is signed and content-rich (e.g., "I expected A but got nothing" vs "I expected B but got nothing").
- **Script**: `scripts/run_individual_decoding.py`.

### Within-Trial Rigorous Null - 54 Tests
- `[Omit Interval]` vs. `[Preceding Delay]` (e.g., `p4` vs `d3` in `AAAX`)
- **Significance**: Since both windows are physically Gray, any above-chance decoding proves the neural response is internally generated by the temporal expectation of the stimulus.
- **Script**: `scripts/run_figure_03_classification_neural.py`.

## 📊 Summary of Interpretation
| Test Outcome | Neural Mechanism |
| :--- | :--- |
| **FX Decoding > Chance** | Tonic Contextual Expectation (State-Space Shift) |
| **Omit vs. Delay > Chance** | Temporal Prediction Error (Phasic Surprise) |
| **Omit A vs. Omit B > Chance** | Contextual Prediction Error (Identity-Specific Surprise) |
| **Oddball > Repeated Accuracy** | Surprise Quenching / Repetition Suppression |
