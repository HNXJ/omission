## Tech Stack

- **Styling:** Custom CSS and SCSS

- **Interactivity:** jQuery, AOS (Animate on Scroll)
- **Special Features:** Spotify embeds, GitHub project links, Python tutorial hosting

## Key Files
- `index.html`: Home page, contains the "Cool projects" and "Musique!" sections.
- `about.html`: About page, education, skills, and "Purpose of my life".
- `contact.html`: Contact information and social links.
- `mindapp.html`: Dedicated page for the MindApp project.
- `css/style.css`: Main stylesheet for custom adjustments.

## Content Management Patterns
- **Adding Projects:** Add a new `<a>` tag with a `<p> </p>` spacer inside the `.site-section-heading` div in `index.html`.
- **Social Links:** Found in the `<footer>` section of all HTML pages.
- **Hero Images:** Background images are defined in the `.site-blocks-cover` div inline style (e.g., `images/winter.jpg` or `images/yellow.jpg`).
