# paper-architecture

## Purpose
Figure-First manuscript protocol: figure manifests drive Methods/Results drafting, 1:1 code-to-text correspondence, Main vs Supplementary organization.

## Workflow
1. Create `docs/figures/FXXX_manifest.md` per figure
2. Lock methodology (exact scripts, filters, windows)
3. Log observations immediately post-analysis
4. Compile Results/Methods from manifests
5. Verify Madelane Golden Dark aesthetic compliance

## Manifest Template
```markdown
# Figure Manifest: FXXX_Title
## Intent — biological question
## Methodology — exact script, window, stats
## Observations — key findings with p-values
## Caption — publication-ready text
```
