# Neuroscience Writing: Reference & DOI Integrity

Ensuring citation accuracy and standardized formatting for systems neuroscience.

## 1. Citation Standards
- **Primary Source**: Always cite the original meta-analysis or experimental study (e.g., Mulvey et al., 2025).
- **Tool Citation**: Formally cite the software libraries used (`Jaxley`, `JAX`, `PyNWB`).
- **Standard format**: Author (Year) or (Author, Year) depending on the journal.

## 2. DOI Management
- **Verification**: Every reference must include a valid DOI.
- **Extraction**: If a DOI is missing in the PDF metadata, use `google_web_search` to find the permanent URI.
- **Linking**: Format DOIs as active hyperlinks in the final PDF/HTML export.

## 3. Biblical Integrity (BibTeX)
- Maintain a centralized `.bib` file in the `AAE` or `HNXJ` repository.
- Ensure all entries include the `doi` field.
- Categorize by theme (e.g., `modeling`, `clinical`, `connectivity`).
