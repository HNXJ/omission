# Omission Frontend Dashboard

This dashboard visualizes the core figures generated by the Omission predictive coding pipeline. It dynamically serves Plotly HTML figures directly from `outputs/oglo-8figs` and provides an interactive gallery grid for high-fidelity data inspection.

## Scope & Content
- Visualizes figures from the 15-step analytical pipeline.
- Surfaces Markdown progress reports for daily tracking.
- Syncs automatically with `manifest.json`.

## Required Files
- `outputs/oglo-8figs/*`: Generated HTML figures.
- `manifest.json`: Dynamic registry of available figures.

## Usage
To start the dashboard locally:
```bash
npm install
npm run dev
```

Then navigate to `http://localhost:5173/` in your browser.
