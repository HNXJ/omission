# Dashboard

Professional React/Vite frontend for the Omission project.

## Workflow
- **Open/View Dashboard**:
  1. Navigate to `D:\drive\omission\dashboard`.
  2. Run `npm install` (if dependencies are missing).
  3. Run `npm run dev` (possibly in the background) to serve the application.
- **Functionality**:
  - Visualizes figures from `outputs/oglo-8figs`.
  - Displays progress reports.
