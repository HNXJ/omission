# Neuroscience Writing: Abstract Crafting (The Mini-Paper)

Structuring high-impact summaries for biophysical research.

## 1. Five-Sentence Structure
1. **Background**: One sentence defining the biophysical role of interneurons in ScZ.
2. **Gap**: One sentence highlighting the lack of link between cellular density and functional motifs.
3. **Methods**: One sentence describing the multi-area Jaxley model and AGSDR optimization.
4. **Findings**: Two sentences reporting the key result (e.g., "Hierarchical PV deficits triggered a cascade of Gamma desynchronization...").
5. **Impact**: One sentence on how this informs future therapeutics or diagnostics.

## 2. Quantitative Precision
- Include specific effect sizes or percentage changes (e.g., "40% reduction in PV conductance").
- Mention specific frequency bands (e.g., "38-42 Hz Gamma").
- State the model scale (e.g., "300-neuron hierarchical super-network").
