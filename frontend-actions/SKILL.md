# Frontend Actions Skill

This skill provides context and workflows for managing the HNXJ personal website and building neuroscience-focused web applications.

## 1. Portfolio Management (hnxj.github.io)
Maintain the HNXJ portfolio by adding new projects and updating educational milestones.

### Common Workflows
- **Adding a New Project**: Append `<a>` links and `<p>` spacers in the project section of `index.html`.
- **Updating Personal Info**: Edit `about.html` for education, awards, and technical skills.
- **Deployment**: Use `gh` (GitHub CLI) for `git add`, `git commit`, and `git push`.

## 2. Neuroscience Webapps (Python)
Building interactive dashboards for neurophysiology and AI models.

### Recommended Frameworks
- **Streamlit**: Best for rapid prototyping of data analysis tools. Use for LFP/Spike visualizations.
- **Plotly Dash**: Better for complex, production-ready neuro-dashboards with deep customization.

### Design Principles
- **Visualization Focus**: Prioritize interactive voltage traces, spectrograms, and raster plots.
- **Interactivity**: Add sliders for simulation parameters (e.g., `gAMPA`, `gGABAa`) to see real-time effects on network activity.
