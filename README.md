## Overview
A Python-based analysis suite for neuroscience data from visual omission paradigms.

## Features
- LFP processing and analysis pipeline.
- Spike train analysis and population coding tools.
- Behavioral data processing (eye-tracking, pupilometry).

## Installation

```bash
pip install omission
```

Or from source:
```bash
git clone https://github.com/HNXJ/omission.git
cd omission
pip install -e .
```
